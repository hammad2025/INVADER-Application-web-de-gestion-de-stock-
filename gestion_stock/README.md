# INVADER - Système de Gestion de Stock

Application web de gestion de stock pour le INVADER permettant de gérer les produits, catégories, fournisseurs et inventaires.

## Fonctionnalités

- Gestion complète des produits avec état de stock
- Gestion des catégories de produits
- Gestion des fournisseurs 
- Suivi des inventaires
- Interface responsive et moderne
- Design élégant avec thème vert émeraude

## Technologies

- PHP 7.4+
- MySQL 5.7+
- Bootstrap 5
- JavaScript (ES6)
- HTML5/CSS3

## Installation

1. Cloner le repository
2. Créer une base de données MySQL nommée `gestion_stock`
3. Importer le fichier `gestion_stock.sql`
4. Configurer la connexion dans `config/database.php`
5. Démarrer un serveur web (ex: MAMP)

## Structure du projet
